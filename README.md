# 🌿 Nature Gallery

A visually stunning **Nature Gallery Web App** that showcases beautiful landscapes and natural scenes with a clean and responsive user interface.

---

## 🚀 Features

* 🌄 Beautiful collection of nature images
* 📱 Fully responsive design (mobile + desktop)
* ⚡ Fast and lightweight UI
* 🎨 Clean and modern layout
* 🔍 Easy navigation and browsing

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/RahulSAxena2628/Naturegallary.git
```

2. Navigate to the project:

```bash
cd Naturegallary
```

3. Open in browser:

```bash
open index.html
```
